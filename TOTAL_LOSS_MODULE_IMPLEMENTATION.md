**파일**: `d:\CascadeProjects\blocks\total_loss_simulator.py`

#### 핵심 기능

**A. 전략 계산 엔진** (`calculate_total_loss_strategy`)
- 입력값: 차량 가액, 예상 수리비, 잔존물 예상 매각가
- 출력 로직:
  - **Case A (수리비 > 가액)**: "추정 전손 대상입니다. 가액 100% 수령 가능."
  - **Case B (수리비 80~100%)**: "임의 전손 협의 가능 구간입니다. 협의 가능성 높음."
  - **Case C (수리비 60~80%)**: "임의 전손 협의 가능 구간입니다. 협의 가능성 보통."
  - **Case D (수리비 < 60%)**: "수리 후 사용 권장입니다. 협의 어려움."

**B. 비교 분석**
- 전손 시 수령액 vs 미수선 보험금(수리비의 80%) + 잔존물 매각액
- 잔존물 차주 보유 협의 시 예상 수령액 (실제 매각가 120% 가정)
- 3가지 전략 중 최대 수익 전략 자동 추천

**C. 실행 단계 자동 생성**
- 추천 전략에 따라 4~5단계의 구체적인 실행 단계 제시
- 예시: "1️⃣ 보험사에 '미수선 처리' 의사 통보 → 2️⃣ 미수선 보험금 수령 → 3️⃣ 사고 차량 매각 → 4️⃣ 총 수령액 확인"

## 📊 핵심 계산 로직

### 수리비 비율 계산
```python
repair_ratio = (repair_cost / vehicle_value * 100)
```

### 미수선 보험금 계산
```python
unrepaired_insurance = int(repair_cost * 0.8)
unrepaired_amount = unrepaired_insurance + salvage_value
```

### 잔존물 차주 보유 협의 시 예상 수령액
```python
estimated_actual_salvage = int(salvage_value * 1.2)  # 실제 매각가는 20% 높다고 가정
salvage_negotiation_amount = (vehicle_value - salvage_value) + estimated_actual_salvage
```

### 최적 전략 선택
```python
strategies = {
    "전손 처리": total_loss_amount,
    "미수선 + 매각": unrepaired_amount,
    "잔존물 차주 보유 협의": salvage_negotiation_amount
}
recommended_strategy = max(strategies, key=strategies.get)
```
