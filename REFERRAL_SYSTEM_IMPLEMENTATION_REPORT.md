- 신규 회원 가입 후 7일 경과

#### 2-2. Python 함수: check_and_grant_pending_rewards

## 🔧 기술 스택

### 데이터베이스
- **PostgreSQL (Supabase)**: 메인 데이터베이스
- **RPC Functions**: 서버 사이드 로직 (보안, 트랜잭션)
- **RLS (Row Level Security)**: 데이터 접근 제어

### 백엔드
- **Python 3.12**: 비즈니스 로직
- **Supabase Client**: DB 연동
- **Pandas**: 데이터프레임 처리

### 프론트엔드
- **Streamlit**: UI 프레임워크
- **HTML/CSS**: 커스텀 스타일링
- **Session State**: 실시간 상태 관리

1. 마이페이지 > "🎁 추천인 리워드" 탭 선택
2. 통계 및 리워드 내역 확인
3. 추천인 정보 수정 가능

### 코인으로 구독 연장하기
1. 마이페이지 > "⚙️ 구독 관리" 탭 선택
2. 현재 보유 코인 및 필요 코인 확인
3. "🪙 120코인으로 1개월 연장하기" 버튼 클릭
4. 성공 시 축하 메시지 및 갱신일 확인
