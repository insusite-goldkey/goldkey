# 자동차 보험 전손 처리 실무 가이드

## 1. 전손의 구분 및 판단 기준

### 1.1 절대전손 (Actual Total Loss)
**정의**: 차량이 물리적으로 완전히 파괴되어 수리가 불가능하거나 원상복구가 사실상 불가능한 상태

**판단 기준**:
- 차량 골격(프레임)이 심각하게 손상되어 안전성 확보 불가
- 화재로 인한 전소(全燒)
- 침수로 인한 엔진 및 전장계통 완전 손상
- 교통사고로 인한 차체 절단 또는 심각한 변형

**보험 처리**:
- 차량 가액의 100% 보상
- 잔존물은 보험사가 소유권 취득
- 별도 협의 없이 자동 전손 처리

### 1.2 추정전손 (Constructive Total Loss)
**정의**: 수리는 가능하나 수리비가 차량 가액을 초과하는 경우

**판단 기준**:
- **수리비 > 차량 가액(시가)**
- 예시: 시가 1,500만 원 차량의 수리비가 1,800만 원으로 산정된 경우

**보험 처리**:
- 차량 가액의 100% 보상
- 차주는 잔존물 소유권을 보험사에 양도
- 보험사는 잔존물을 경매 또는 매각하여 손해액 회수

**실무 팁**:
- 수리비 견적서는 반드시 정비업체 공식 견적서 제출
- 부품 가격은 순정 부품 기준으로 산정
- 공임비는 지역별 표준 공임 기준 적용

### 1.3 임의전손 (경제적 전손, Economic Total Loss)
**정의**: 수리비가 차량 가액의 일정 비율(통상 60~100%) 이상이지만 차량 가액을 초과하지 않아, 차주와 보험사가 협의하여 전손으로 처리하는 경우

**판단 기준**:
- **차량 가액의 60% ≤ 수리비 < 차량 가액**
- 예시: 시가 2,000만 원 차량의 수리비가 1,400만 원(70%)인 경우

**협의 가능 구간**:
- 수리비 60~80%: 협의 가능성 보통
- 수리비 80~100%: 협의 가능성 높음
- 수리비 100% 이상: 추정전손으로 자동 전환

**보험 처리**:
- 차주와 보험사 간 합의 필수
- 차량 가액 또는 합의된 금액 지급
- 잔존물 처리 방식은 협의에 따라 결정

## 2. 임의전손 유도 전략 (설계사 핵심 노하우)

### 2.1 잔존물 매각 협의 전략

**전략 1: 잔존물 차주 보유 + 차액 정산 방식**

**프로세스**:
1. 보험사에 "잔존물을 차주가 직접 보유하고, 차량 가액에서 잔존물 가치를 차감한 금액을 보험금으로 수령하겠다"고 제안
2. 잔존물 가치 산정: 보험사 또는 제3자 감정평가 의뢰
3. 보험금 = 차량 가액 - 잔존물 가치
4. 차주는 잔존물을 직접 매각하여 추가 수익 확보

**예시**:
- 차량 가액: 2,000만 원
- 수리비: 1,600만 원 (80%)
- 잔존물 감정가: 500만 원
- 보험금 수령: 2,000만 원 - 500만 원 = 1,500만 원
- 차주가 잔존물을 700만 원에 직접 매각 성공
- **실제 수령액: 1,500만 원 + 700만 원 = 2,200만 원** (차량 가액 대비 +200만 원)

**전략 2: 미수선 처리 + 잔존물 매각 조합**

**프로세스**:
1. 보험사에 "수리하지 않고 미수선 보험금(수리비의 80~90%)을 수령하겠다"고 통보
2. 미수선 보험금 수령 후 차량을 그대로 매각
3. 차량 매각가 + 미수선 보험금 = 총 수령액

**예시**:
- 차량 가액: 2,000만 원
- 수리비: 1,600만 원
- 미수선 보험금: 1,600만 원 × 80% = 1,280만 원
- 사고 차량 매각가: 800만 원 (중고차 시장 거래)
- **실제 수령액: 1,280만 원 + 800만 원 = 2,080만 원** (차량 가액 대비 +80만 원)

**전략 3: 보험사 주도 전손 처리 + 잔존물 경매 참여**

**프로세스**:
1. 보험사가 전손 처리하여 차량 가액 100% 지급
2. 보험사는 잔존물을 경매에 출품
3. 차주 또는 지인이 경매에 참여하여 낙찰
4. 낙찰가가 시장가보다 낮으면 추가 이익 발생

**주의사항**:
- 경매 참여 시 본인 명의 직접 입찰은 보험사기 의심 가능
- 제3자(가족, 지인) 명의로 참여 권장
- 낙찰 후 재매각 시 세금 문제 검토 필수

### 2.2 보험사 설득 포인트

**차주 입장에서 강조할 사항**:
1. **수리 후 차량 가치 하락(격락손해)**: "수리해도 사고 이력으로 인해 차량 가치가 30~40% 하락합니다. 전손 처리가 합리적입니다."
2. **수리 기간 중 대차료 부담**: "수리 기간이 2~3개월 소요되면 대차료가 추가로 발생합니다. 전손 처리가 보험사에도 유리합니다."
3. **수리 품질 불확실성**: "대형 사고 차량은 수리 후에도 안전성이 보장되지 않습니다. 전손 처리가 안전합니다."

**보험사 입장에서 유리한 점**:
1. 수리비 지급보다 차량 가액 지급이 총 손해액이 적을 수 있음
2. 잔존물 매각으로 손해액 일부 회수 가능
3. 대차료 장기 지급 부담 회피

## 3. 차주 혜택 및 권리

### 3.1 취등록세 지원 (신규 차량 구입 시)

**지원 대상**:
- 전손 처리 후 6개월 이내 동일 차종 또는 유사 차종 신규 구입 시
- 보험사에 따라 지원 범위 상이

**지원 내용**:
- 취득세 + 등록세 전액 또는 일부 지원
- 통상 차량 가격의 7~10% 수준
- 예시: 3,000만 원 신차 구입 시 약 210~300만 원 지원

**신청 방법**:
1. 전손 보험금 수령 시 보험사에 "신차 구입 예정"임을 사전 통보
2. 신차 구입 후 차량등록증, 세금 납부 영수증 제출
3. 보험사 심사 후 취등록세 지원금 별도 지급

**주의사항**:
- 보험사별 지원 정책 상이 (약관 확인 필수)
- 일부 보험사는 지원 제외 (저가 보험사 주의)
- 신차가 아닌 중고차 구입 시 지원 불가

### 3.2 대차료 인정 기간

**대차료 정의**:
- 사고 차량 수리 또는 전손 처리 기간 중 렌터카 등 대체 차량 이용 비용

**인정 기간**:
- **수리 시**: 실제 수리 기간 (통상 10~30일)
- **전손 시**: 보험금 지급 결정일까지 (통상 7~14일)
- **협의 지연 시**: 최대 30일까지 인정 (보험사 귀책 사유 시 연장 가능)

**대차료 산정 기준**:
- 소형차: 일 3~5만 원
- 중형차: 일 5~8만 원
- 대형차/수입차: 일 8~15만 원

**실무 팁**:
- 렌터카 계약 전 보험사에 사전 승인 요청
- 렌터카 영수증 반드시 보관
- 과도한 고급 차량 렌트 시 일부 미인정 가능

### 3.3 격락손해금 청구

**격락손해 정의**:
- 사고 차량을 수리했음에도 불구하고 사고 이력으로 인해 차량 가치가 하락하는 손해

**청구 가능 조건**:
1. 차량 연식 3년 이내 또는 주행거리 6만 km 이내
2. 차량 골격(프레임) 손상 또는 주요 부품 교체
3. 수리비가 차량 가액의 30% 이상

**격락손해금 산정**:
- 통상 차량 가액의 10~30% 수준
- 예시: 시가 3,000만 원 차량의 격락손해금 300~900만 원

**청구 방법**:
1. 수리 완료 후 보험사에 격락손해금 청구서 제출
2. 제3자 감정평가 의뢰 (비용은 보험사 부담)
3. 감정 결과에 따라 격락손해금 지급

**주의사항**:
- 전손 처리 시에는 격락손해금 청구 불가 (이미 차량 가액 100% 수령)
- 미수선 처리 시에도 청구 불가
- 수리 후에만 청구 가능

### 3.4 보험료 환급 및 승계

**보험료 환급**:
- 전손 처리 시 남은 보험 기간에 대한 보험료 일할 환급
- 예시: 1년 보험료 100만 원, 6개월 사용 후 전손 → 50만 원 환급

**보험 승계**:
- 신차 구입 시 기존 보험을 신차로 승계 가능
- 무사고 할인 등급 유지
- 보험료 차액 정산 (신차가 고가인 경우 추가 납부)

## 4. 전손 처리 시 주의사항

### 4.1 압류 및 저당 해지

**확인 사항**:
- 차량에 압류, 가압류, 저당권 설정 여부 확인 (등록원부 조회)
- 압류/저당 미해지 시 보험금 수령 불가

**해지 절차**:
1. 채권자에게 연락하여 채무 상환 또는 협의
2. 해지 서류 발급 (압류 해제 확인서, 저당권 말소 동의서)
3. 관할 시·군·구청에 해지 등록

### 4.2 하이패스 카드 및 개인 소지품 회수

**회수 항목**:
- 하이패스 카드 (미회수 시 통행료 계속 청구)
- 블랙박스 SD 카드 (사고 영상 증거 보존)
- 내비게이션, 차량용 방향제, 핸드폰 거치대 등 개인 소지품

**회수 시기**:
- 보험사에 차량 인도 전 반드시 회수
- 인도 후에는 회수 불가 (보험사 책임 없음)

### 4.3 자동차세 환급

**환급 대상**:
- 전손 처리 후 차량 말소 등록 시 남은 기간에 대한 자동차세 환급

**환급 신청**:
1. 차량 말소 등록 (보험사 또는 차주가 직접 처리)
2. 관할 시·군·구청에 자동차세 환급 신청
3. 환급금은 차주 계좌로 입금

## 5. 전손 처리 준비 서류

### 5.1 개인 차주

**필수 서류**:
1. **자동차 매도용 인감증명서** (발급일로부터 3개월 이내)
   - ⚠️ **매수인 정보 기재 필수**: 보험사명, 사업자등록번호 반드시 기재
   - 용도: "자동차 매도용"으로 명시
2. 자동차등록증 원본
3. 신분증 사본 (주민등록증 또는 운전면허증)
4. 통장 사본 (보험금 입금 계좌)

**선택 서류** (해당 시):
- 압류/저당 해지 서류
- 자동차세 완납 증명서

### 5.2 개인사업자 차주

**필수 서류**:
1. **자동차 매도용 인감증명서** (사업자 인감 사용)
   - ⚠️ **매수인 정보 기재 필수**: 보험사명, 사업자등록번호 반드시 기재
2. 사업자등록증 사본
3. 자동차등록증 원본
4. 대표자 신분증 사본
5. 사업자 통장 사본

**선택 서류** (해당 시):
- 사업자 폐업 시 폐업 사실 증명서
- 압류/저당 해지 서류

### 5.3 법인 차주

**필수 서류**:
1. **자동차 매도용 법인 인감증명서** (발급일로부터 3개월 이내)
   - ⚠️ **매수인 정보 기재 필수**: 보험사명, 사업자등록번호 반드시 기재
2. 법인 인감도장 날인된 위임장 (보험사 양식)
3. 법인등기부등본 (발급일로부터 3개월 이내)
4. 사업자등록증 사본
5. 자동차등록증 원본
6. 대표이사 신분증 사본
7. 법인 통장 사본

**선택 서류** (해당 시):
- 이사회 결의서 (고가 차량 또는 내규 상 필요 시)
- 압류/저당 해지 서류

### 5.4 공통 주의사항

**인감증명서 발급 시 반드시 확인**:
- ⚠️ **매수인(보험사) 정보 기재란에 보험사명과 사업자등록번호를 반드시 기재**
- 백지 인감증명서는 보험사에서 접수 거부
- 용도: "자동차 매도용" 명시 필수

**서류 제출 시 체크리스트**:
- [ ] 인감증명서 유효기간 확인 (3개월 이내)
- [ ] 매수인 정보 기재 여부 확인 (빨간색 강조)
- [ ] 자동차등록증 원본 여부 확인
- [ ] 압류/저당 해지 여부 확인
- [ ] 하이패스 카드 회수 여부 확인
- [ ] 개인 소지품 회수 여부 확인
- [ ] 보험료 환급 신청 여부 확인

## 6. 전손 vs 수리 vs 미수선 비교표

| 구분 | 전손 처리 | 수리 후 사용 | 미수선 처리 |
|------|----------|-------------|------------|
| **보험금 수령액** | 차량 가액 100% | 수리비 100% | 수리비의 80~90% |
| **차량 소유권** | 보험사 이전 | 차주 유지 | 차주 유지 |
| **추가 수익 가능성** | 잔존물 매각 시 가능 | 없음 | 차량 매각 시 가능 |
| **격락손해금 청구** | 불가 | 가능 | 불가 |
| **대차료 인정** | 7~14일 | 10~30일 | 없음 |
| **취등록세 지원** | 가능 (신차 구입 시) | 없음 | 없음 |
| **보험료 환급** | 가능 | 없음 | 없음 |
| **차량 사용 가능 여부** | 불가 | 가능 | 가능 (수리 안 함) |

## 7. 실무 사례 및 최적 전략

### 사례 1: 수리비 85% 구간 (임의전손 협의 최적)

**상황**:
- 차량 가액: 2,500만 원
- 수리비: 2,125만 원 (85%)
- 잔존물 예상 매각가: 800만 원

**전략 분석**:
1. **전손 처리**: 2,500만 원 수령 (잔존물은 보험사 소유)
2. **미수선 + 매각**: 2,125만 원 × 80% = 1,700만 원 + 800만 원 = 2,500만 원
3. **잔존물 차주 보유 협의**: 2,500만 원 - 800만 원 = 1,700만 원 + 실제 매각가 900만 원 = 2,600만 원

**최적 전략**: 잔존물 차주 보유 협의 (보험사 설득 필요)

### 사례 2: 수리비 65% 구간 (미수선 처리 유리)

**상황**:
- 차량 가액: 3,000만 원
- 수리비: 1,950만 원 (65%)
- 사고 차량 예상 매각가: 1,500만 원

**전략 분석**:
1. **전손 처리 협의**: 3,000만 원 수령 (협의 난이도 높음)
2. **미수선 + 매각**: 1,950만 원 × 80% = 1,560만 원 + 1,500만 원 = 3,060만 원
3. **수리 후 사용**: 1,950만 원 수리비 지급 + 격락손해금 300만 원 = 2,250만 원 (차량 가치 하락 고려 시 손해)

**최적 전략**: 미수선 + 매각 (전손 협의보다 확실하고 수익 높음)

### 사례 3: 수리비 95% 구간 (추정전손 임박)

**상황**:
- 차량 가액: 1,800만 원
- 수리비: 1,710만 원 (95%)
- 잔존물 예상 매각가: 600만 원

**전략 분석**:
1. **전손 처리**: 1,800만 원 수령 (협의 용이)
2. **미수선 + 매각**: 1,710만 원 × 80% = 1,368만 원 + 600만 원 = 1,968만 원
3. **수리 후 사용**: 1,710만 원 수리비 지급 (차량 가치 하락으로 손해)

**최적 전략**: 미수선 + 매각 (전손보다 168만 원 추가 수익)

## 8. 보험사 협의 시 유의사항

### 8.1 협의 타이밍

**최적 타이밍**:
- 수리비 견적서 확정 직후
- 보험사 손해사정 완료 전
- 차량 인도 전

**피해야 할 타이밍**:
- 이미 수리 착수 후 (전손 협의 불가)
- 차량을 보험사에 인도한 후 (협상력 상실)

### 8.2 협의 전략

**강경 전략** (수리비 80% 이상):
- "수리 후 차량 가치 하락이 크므로 전손 처리가 합리적입니다."
- "수리 기간 중 대차료 부담을 고려하면 보험사에도 전손이 유리합니다."
- "잔존물 매각으로 보험사 손해액을 줄일 수 있습니다."

**유연 전략** (수리비 60~80%):
- "잔존물을 제가 직접 보유하고 차액만 받겠습니다."
- "미수선 처리로 진행하겠습니다." (협상 카드로 활용)

### 8.3 보험사 거부 시 대응

**거부 사유별 대응**:
1. "수리비가 차량 가액을 초과하지 않습니다."
   - → 미수선 처리 + 매각 전략으로 전환
2. "잔존물 가치가 너무 높습니다."
   - → 제3자 감정평가 의뢰 제안
3. "약관상 임의전손 조항이 없습니다."
   - → 금융감독원 민원 제기 (최후 수단)

## 9. 법적 근거 및 참고 자료

### 9.1 관련 법령

- **상법 제682조** (보험가액): 보험 목적의 가액은 보험사고 발생 시의 가액으로 한다.
- **상법 제678조** (전부멸실과 일부멸실): 보험의 목적의 전부가 멸실한 때에는 보험금액의 전부를 지급한다.
- **자동차손해배상 보장법 시행령 제3조** (보험금 지급 기준)

### 9.2 금융감독원 분쟁조정 기준

- 수리비가 차량 가액의 80% 이상인 경우 전손 처리 권고

- 보험사의 일방적 전손 거부는 부당행위로 판단 가능
