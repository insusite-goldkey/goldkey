## 2️⃣ 금융공학 엔진 상세

### A. 복리 산식 (Financial Engineering Formulas)

#### 1️⃣ **적립기 미래가치 (Future Value)**
```
FV = PV(1 + r)^n + PMT × [(1 + r)^n - 1] / r

where:
  PV  = 현재가치 (초기 투자금)
  PMT = 월 납입액
  r   = 월 수익률 (연 수익률 ÷ 12)
  n   = 총 납입 개월수 (납입 기간 × 12)
```

**특징**:
- 월 복리 기준 (단리 금지)
- 초기 투자금 + 정기 납입 모두 반영
- 실제 금융상품 수익률 정확 반영

#### 2️⃣ **수령기 월 연금액 (Annuity Payment)**

**표준 방식 (자산 고갈)**:
```
PMT = FV × r / [1 - (1 + r)^(-n)]

where:
  FV = 적립 총액
  r  = 월 수익률
  n  = 총 수령 개월수
```

**트리니티 4% 모드 (원금 보존)**:
```
PMT = FV × 0.04 / 12

where:
  FV = 적립 총액
  0.04 = 연 4% 안전 인출률
```

**특징**:
- 미상환 잔액 복리 방식 (수령 중에도 남은 돈이 증식)
- 트리니티 연구 기반 안전 인출률 옵션
- 자산 고갈 vs 원금 보존 선택 가능

#### 3️⃣ **물가상승률 반영 미래 생활비**
```
Future Cost = Current Cost × (1 + inflation_rate)^years

where:
  Current Cost = 현재 최소 생활비
  inflation_rate = 물가상승률 (3~5%)
  years = 은퇴까지 기간
```

**특징**:
- 보수적(3%), 중립(4%), 공격적(5%) 시나리오
- 30년 후 필요 생활비 = 현재의 2~4배
- "공포 수치" 시각화로 추가 가입 명분 창출
