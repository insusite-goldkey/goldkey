# 전문 금융 컨설턴트 언어 가이드라인

**작성일**: 2026-03-31  
**목적**: 전투적/번역투/난해한 한자어 배제, 전문 금융 컨설턴트 톤 확립  
**적용 범위**: 모든 UI 텍스트, 시스템 메시지, 리포트, 분석 결과

---

## 🎯 핵심 원칙

### 언어 철학
> "전문성은 유지하되, 고객이 상황을 그릴 수 있는 쉬운 언어를 사용하라."

### 3대 배제 원칙

#### ❌ 1. 전투적 표현 배제
- 공격, 방어, 무기, 전략, 전술 등 군사 용어 지양
- 고객의 자산을 **안전하게 관리(보전)**한다는 톤 유지

#### ❌ 2. 번역투 배제
- 직역체 문장 구조 지양
- 자연스러운 한국어 어순 사용

#### ❌ 3. 난해한 한자어 배제
- 임계점 → 한계 지점
- 비가역적 → 돌이키기 어려운
- 상황이 그려지는 쉬운 전문 용어 선택

## 📚 용어 사전

### ❌ 금지 용어 → ✅ 권장 용어

#### 전투적 표현

| ❌ 금지 | ✅ 권장 | 설명 |
|---------|---------|------|
| 방어 | 보전, 완충 | 고객 자산을 안전하게 관리 |
| 무기 | 수단, 도구 | 문제 해결 방법 |
| 전략 | 계획, 방안 | 목표 달성 방법 |
| 전술 | 실행 방법 | 구체적 접근법 |
| 공격적 | 적극적 | 능동적 태도 |
| 타겟팅 | 집중 분석 | 특정 영역 분석 |
| 비수 | 핵심 근거 | 설득력 있는 논리 |
| 압도 | 명확한 제시 | 분명한 전달 |

#### 번역투 표현

| ❌ 금지 | ✅ 권장 | 설명 |
|---------|---------|------|
| ~에 대한 분석 | ~분석 | 간결한 표현 |
| ~를 통해 | ~로 | 자연스러운 조사 |
| ~에 있어서 | ~에서 | 불필요한 격식 제거 |
| ~함으로써 | ~해서 | 쉬운 연결어 |
| 시각화했습니다 | 직관적 리포트를 생성했습니다 | 결과 중심 표현 |

#### 난해한 한자어

| ❌ 금지 | ✅ 권장 | 설명 |
|---------|---------|------|
| 임계점 | 한계 지점 | 상황이 그려지는 표현 |
| 비가역적 | 돌이키기 어려운 | 구체적 상황 설명 |
| 불가피 | 피할 수 없는 | 쉬운 표현 |
| 필연적 | 반드시 일어나는 | 명확한 의미 |
| 상쇄 | 균형 맞춤, 보완 | 이해하기 쉬운 표현 |
| 잠재적 | 숨어있는, 앞으로 생길 수 있는 | 상황 설명 |
| 촉발 | 시작, 발생 | 간단한 표현 |
| 귀결 | 결과 | 직관적 표현 |

## 💬 문장 톤 가이드

### 1. 분석 결과 전달

#### ❌ 전투적/번역투
```
"제가 분석한 결과, 이 증권은 물가 상승률이 반영되지 않은 박제된 계약입니다.
130% 플랜으로 즉시 업그레이드가 필요합니다."
```

#### ✅ 금융 컨설턴트 톤
```
"분석 결과, 이 증권은 물가 상승률이 반영되지 않아 
실질 보장액이 감소한 상태입니다.
현재 가치에 맞춰 보장 금액을 조정하시는 것을 권장드립니다."
```

### 2. 리스크 경고

#### ❌ 전투적/번역투
```
"위 리스크들은 CEO 유고 시 즉시 현실화되는 항목들입니다.
현금 확보 없이는 사업 계속성(Going Concern)이 불가능합니다."
```

#### ✅ 금융 컨설턴트 톤
```
"이러한 리스크는 경영자 부재 시 바로 현실이 됩니다.
사업을 계속 운영하기 위해서는 충분한 현금 확보가 필수입니다."
```

### 3. 솔루션 제시

#### ❌ 전투적/번역투
```
"이 전략은 상속세 재원 확보와 동시에 사업 계속성을 보장하는 
유일한 방어책입니다."
```

#### ✅ 금융 컨설턴트 톤
```
"이 방안은 상속세 납부 자금을 마련하면서 
사업도 안정적으로 이어갈 수 있는 효과적인 해결책입니다."
```

### 4. 데이터 시각화

#### ❌ 번역투
```
"데이터를 시각화했습니다."
```

#### ✅ 금융 컨설턴트 톤
```
"한눈에 파악 가능한 직관적 리포트를 생성했습니다."
```

## 📊 리포트 작성 가이드

### Executive Summary (경영진 요약)

#### ❌ 전투적/한자어 과다
```
제가 분석한 결과, 현재 고객님의 총 리스크 규모는 450,000,000원에 달합니다.
이는 상속세 300,000,000원과 추가 리스크 2건을 합산한 금액입니다.
즉시 대응이 필요한 상황입니다.
```

#### ✅ 금융 컨설턴트 톤
```
분석 결과, 현재 대표님께서 준비하셔야 할 금액은 총 4억 5천만원입니다.
이는 상속세 3억원과 사업 운영 자금 2건을 합친 금액입니다.
지금부터 단계적으로 준비하시는 것을 권장드립니다.
```

### Risk Assessment (리스크 평가)

#### ❌ 전투적/한자어 과다
```
⚡ 제가 경고드립니다
위 리스크들은 CEO 유고 시 즉시 현실화되는 항목들입니다.
현금 확보 없이는 사업 계속성(Going Concern)이 불가능합니다.
```

#### ✅ 금융 컨설턴트 톤
```
💡 주의하실 점
이러한 항목들은 경영자 부재 시 바로 발생하는 비용입니다.
충분한 현금이 없다면 사업을 계속 운영하기 어려울 수 있습니다.
```

### Strategic Recommendation (전략적 권고사항)

#### ❌ 전투적/한자어 과다
```
💡 제가 제시하는 솔루션
핵심 전략: CEO 경영인정기보험 + 유동성 확보 플랜
이 전략은 상속세 재원 확보와 동시에 사업 계속성을 보장하는 유일한 방어책입니다.
```

#### ✅ 금융 컨설턴트 톤
```
💡 권장 해결 방안
핵심 방안: CEO 경영인정기보험 + 운영 자금 확보 계획
이 방법은 상속세 납부 자금을 마련하면서 사업도 안정적으로 이어갈 수 있는 
효과적인 해결책입니다.
```

## 🔤 전문 용어 사용 가이드

### 허용되는 전문 용어

다음 용어들은 금융 업계 표준 용어로 사용 가능:

- **상속세**: 일반적으로 알려진 용어
- **보험료**: 표준 용어
- **보장액**: 표준 용어
- **실손보험**: 표준 용어
- **종신보험**: 표준 용어
- **경영인보험**: 업계 표준 용어

### 설명이 필요한 전문 용어

다음 용어들은 반드시 쉬운 설명과 함께 사용:

- **Going Concern** → "사업 계속성" (사업을 계속 운영하는 것)
- **Trinity Value** → "트리니티 가치 분석" (세 가지 관점에서 본 가치)
- **Pain Point** → "핵심 문제점" (가장 큰 어려움)

## 🔍 검증 체크리스트

모든 텍스트 작성 시 다음 항목을 확인:

### 1. 전투적 표현 제거
- [ ] 방어, 공격, 무기, 전술 등 군사 용어 사용 안 함
- [ ] 보전, 완충, 수단, 계획 등 대체 용어 사용

### 2. 번역투 제거
- [ ] 자연스러운 한국어 어순 사용
- [ ] "~에 대한", "~를 통해" 등 불필요한 표현 제거

### 3. 난해한 한자어 제거
- [ ] 임계점 → 한계 지점
- [ ] 비가역적 → 돌이키기 어려운
- [ ] 상황이 그려지는 쉬운 표현 사용

### 4. 전문성 유지
- [ ] 금융 업계 표준 용어 적절히 사용
- [ ] 구체적 숫자와 근거 제시
- [ ] 객관적이고 신뢰할 수 있는 톤

### 5. 고객 중심
- [ ] 고객이 이해하기 쉬운 표현
- [ ] 상황이 그려지는 구체적 설명
- [ ] 실질적 도움이 되는 정보 제공
