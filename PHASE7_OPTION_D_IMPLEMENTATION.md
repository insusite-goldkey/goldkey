```
🚨 [보험사 현장조사 대처 및 치료 주의사항 안내]

안녕하세요, 홍길동님.
가입하신 지 얼마 되지 않아 사고가 발생하여(근접사고) 보험사에서 통상적인 확인 절차로 손해사정인(조사자)을 파견하게 되었습니다. 너무 걱정하지 마시고 아래 사항만 주의해 주시면 됩니다!

1️⃣ **조사자 미팅 시 대처 방안 (서명 주의!)**

- 동의서 서명 주의: '의무기록 열람 동의서'는 서명 가능하나, '국민건강보험공단 요양급여내역 발급 위임장'이나 '국세청 연말정산 조회 동의서'는 의무 사항이 아니므로 가급적 서명을 보류하시고 저에게 먼저 연락 주세요.

2️⃣ **병원 진료 시 절대 주의사항 (초진 차트 방어)**
- 말조심: "예전부터 아팠다" 또는 "몇 년 전에도 삐끗했다"는 발언 절대 금물
- 명확한 인과관계: 반드시 "이번 사고로 처음 심하게 아프기 시작했다"고 명확히 말씀하세요

3️⃣ **향후 치료 방향**
- 조사 결과가 나올 때까지는 비급여 도수치료나 고가 MRI는 보류될 수 있으니, 우선 급여 물리치료와 약물치료 위주로 진행하시기를 권장합니다.

궁금하신 점이나 조사자 방문 일정이 잡히면 즉시 연락 주세요!
```

## 🎯 주요 안내 내용

### 1️⃣ 조사자 미팅 시 대처 방안

#### 방문 목적
- 가입 전 병력 확인 (고지의무 위반 여부)
- 과거 5년 치 병원 기록 조회

#### 동의서 서명 주의사항

| 동의서 종류 | 서명 가능 여부 | 설명 |
|-----------|--------------|------|
| **의무기록 열람 동의서** | ✅ 서명 가능 | 해당 병원 기록만 열람 |
| **국민건강보험공단 요양급여내역 발급 위임장** | ⚠️ 보류 권장 | 과거 5년 치 전체 병원 기록 조회 |
| **국세청 연말정산 조회 동의서** | ⚠️ 보류 권장 | 의료비 공제 내역 조회 |

**핵심 메시지**: 의무 사항이 아니므로 가급적 서명 보류 후 설계사에게 먼저 연락

---

### 2️⃣ 병원 진료 시 절대 주의사항

#### ❌ 절대 금지 발언
- "사실 예전부터 허리가 좀 뻐근하긴 했는데..."
- "몇 년 전에도 삐끗한 적이 있어요"
- "수년 전부터 요통이 있었어요"

#### ✅ 명확한 인과관계 표현
- "이번에 넘어지면서(Slip down) 처음으로 심하게 아프기 시작했다"
- "이번 사고 전에는 이런 통증이 없었다"
- "사고 직후부터 증상이 시작되었다"

**핵심 메시지**: 의사가 무심코 적은 '수년 전부터 요통 있었음' 한 줄 때문에 보험금 부지급 가능

---

### 3️⃣ 향후 치료 방향

#### 조사 기간 중 권장 치료
- ✅ **급여 물리치료** (보험 적용)
- ✅ **약물치료** (진통제, 소염제)
- ✅ **필수 검사** (X-ray, 기본 MRI)

#### 조사 기간 중 보류 권장
- ⚠️ **비급여 도수치료** (고가)
- ⚠️ **고가 MRI** (3.0T MRI 등)
- ⚠️ **비급여 주사 치료**

**핵심 메시지**: 조사 결과가 나올 때까지 보험금 지급 보류 가능성 있음

## 🚀 사용 방법

### 1. CRM 앱 실행
```powershell
streamlit run crm_app.py --server.port 8503
```

### 2. 근접사고 고객 상담 시나리오

1. **고객 선택** (가입 1~3개월 내 사고 발생 고객)

2. **AI 상담 채팅** 또는 **의무기록 분석** 실행

3. **분석 결과 하단**에 "📱 고객에게 카톡으로 보내기" 섹션 표시

4. **옵션 D 선택**: 🚨 근접사고 및 현장조사 대처 요령

5. **"✨ 요약본 생성"** 버튼 클릭

6. **1-2분 후 요약본 확인** (Gemini 1.5 Pro 생성)

7. **메시지 수정** (필요 시)

8. **복사 또는 공유**:
   - PC: "📋 복사하기" → 카카오톡에 붙여넣기
   - 모바일: "💬 카톡 공유" → 대화방 선택
