## 📋 Executive Summary

### 🎯 **작업 목표**
화재보험의 핵심인 '건물 급수'를 AI가 자동으로 판단하게 하는 전문가 로직을 이식하여, 5:5 대시보드 좌측에서 건물 정보를 입력하면 우측 분석창에서 즉시 급수 판정 결과와 요율 할증률을 확인할 수 있도록 구현.

### ✅ **핵심 성과**
1. **급수 판정 엔진**: 1~4급 자동 분류 (구조/지붕/외벽 기반)
2. **요율 계산 시스템**: 1급 대비 할증률 자동 계산 (0~500%)
3. **5:5 대시보드 UI**: 좌측 입력 + 우측 분석창 (파스텔 테마)
4. **HQ 앱 통합**: policy_scan 탭에 "🏢 건물 급수 산출" 추가
5. **실시간 보험료 계산**: 건물 가액 입력 시 연간/월 보험료 즉시 표시

```
    ↓
[calculate_premium()]
  └─ 건물 가액 × 요율 = 연간 보험료 → 월 보험료
    ↓

  ├─ 급수 배지 (1~4급)
  ├─ 위험도 표시 (낮음/보통/높음/매우높음)
  ├─ 요율 정보 (기본 요율 + 할증률)
  ├─ 예상 보험료 (연간/월)
  └─ 권장사항 + 상세 설명
```

## 2️⃣ 급수 판정 기준 (Fire Insurance Standard)

### A. 1급 (최우수 등급)
**조건**: 기둥/보/바닥이 내화구조(콘크리트, 철근콘크리트) + 지붕이 불연재료

| 항목 | 값 |
|------|-----|
| **기본 요율** | 0.05% |
| **할증률** | 0% (기준) |
| **위험도** | 낮음 ✅ |
| **권장사항** | 최우수 등급입니다. 화재 위험이 매우 낮아 보험료가 가장 저렴합니다. |

**보험료 예시** (건물 가액 5억원):
- 연간 보험료: **250,000원**

| 항목 | 값 |
|------|-----|
| **기본 요율** | 0.08% |
| **할증률** | 60% (1급 대비) |
| **위험도** | 보통 ⚠️ |
| **권장사항** | 양호한 등급입니다. 외벽과 지붕이 불연재료로 화재 위험이 낮습니다. |

**보험료 예시** (건물 가액 5억원):

**조건**: 목재 또는 가연성 외벽

| 항목 | 값 |
|------|-----|
| **기본 요율** | 0.15% |
| **할증률** | 200% (1급 대비) |
| **위험도** | 높음 🔥 |
| **권장사항** | ⚠️ 주의 필요. 외벽이 가연성 재료로 화재 위험이 높습니다. |

**보험료 예시** (건물 가액 5억원):
- 연간 보험료: **750,000원** (1급 대비 +500,000원)
- 월 보험료: **62,500원**

**상세 설명**:
외벽이 목재 또는 가연성재료로 되어 있어 화재 발생 시 빠르게 확산될 위험이 있습니다. 1급 대비 요율이 200% 높으며, 화재 예방 시설(소화기, 스프링클러) 설치를 강력히 권장합니다.

| **할증률** | 500% (1급 대비) |
| **위험도** | 매우높음 🚨 |
| **권장사항** | 🔥 고위험 등급. 가설 건축물 또는 가연성 지붕으로 화재 위험이 매우 높습니다. |

**보험료 예시** (건물 가액 5억원):

**상세 설명**:
구조가 가설구조이거나 지붕이 가연재료로 되어 있어 화재 발생 시 전소 위험이 매우 높습니다. 1급 대비 요율이 500% 높으며, 보험 가입이 거부될 수 있습니다. 건물 구조 개선(외벽 교체, 지붕 불연재료 교체)을 강력히 권장합니다.
