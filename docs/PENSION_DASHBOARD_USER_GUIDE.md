## 3. 단계별 사용법

### 📝 Step 1: 고객 기본 정보 입력

#### 1-1. 나이 정보
```
┌─────────────────────────────────┐
│ 현재 나이: [40] 세              │
│ 은퇴 예정 나이: [65] 세         │
│ 기대 수명: [85] 세              │
│ 월 급여: [400] 만원             │
└─────────────────────────────────┘
```

**입력 팁**:
- 현재 나이: 실제 나이 입력
- 은퇴 나이: 일반적으로 60~65세
- 기대 수명: 보수적으로 85~90세 권장
- 월 급여: 건강보험료 역산 소득 참고

#### 1-2. 생활비 기준 설정
```
┌─────────────────────────────────┐
│ 거주 지역: [광역 ▼]             │
│ 가구 구분: [부부 ▼]             │
└─────────────────────────────────┘
```

**지역별 최소 생활비 (2024년 기준)**:
| 지역 | 단독 | 부부 |
|------|------|------|
| 광역 | 180만원 | 310만원 |
| 중소 | 150만원 | 260만원 |
| 농어촌 | 120만원 | 210만원 |

**선택 가이드**:
- 광역: 서울, 부산, 대구, 인천, 광주, 대전, 울산
- 중소: 수원, 성남, 고양, 용인, 창원 등
- 농어촌: 군 단위 지역

---

### 💰 Step 2: 연금 납입 계획 입력

```
┌─────────────────────────────────┐
│ 월 납입액: [30] 만원            │
│ 예상 연 수익률: [4.5] %         │
└─────────────────────────────────┘
```

**수익률 가이드**:
- **보수적 (3~4%)**: 원리금 보장형, 채권형 펀드
- **중립 (4~5%)**: 혼합형 펀드, 변액연금
- **공격적 (5~7%)**: 주식형 펀드, 해외 투자

**월 납입액 권장**:
- 월 소득의 10~15% 수준
- 세액공제 한도: 연 900만원 (월 75만원)
- 최소 권장: 월 30만원 이상

---

### 🏛️ Step 3: 4층 보장 현황 입력

```
┌─────────────────────────────────┐
│ 국민연금 예상액: [172] 만원/월  │
│ 퇴직연금 예상액: [64] 만원/월   │
│ 주택연금 예상액: [54] 만원/월   │
│ 연 총급여: [5000] 만원          │
└─────────────────────────────────┘
```

**입력 방법**:
1. **국민연금**: 국민연금공단 앱/홈페이지에서 예상 수령액 조회
2. **퇴직연금**: 회사 퇴직연금 담당자 문의 또는 DC형 적립액 확인
3. **주택연금**: 한국주택금융공사 홈페이지 계산기 활용
4. **연 총급여**: 세액공제율 계산용 (원천징수영수증 참고)

**입력 없을 경우**:
- 0원으로 입력 (시스템이 자동으로 처리)
- 국민연금만 있는 경우도 분석 가능

---

### ⚙️ Step 4: 분석 옵션 설정

```
┌─────────────────────────────────┐
│ 물가상승률 시나리오:            │
│ ○ 보수적 (3%)                   │
│ ○ 중립 (4%)                     │
│ ● 공격적 (5%) ← 권장            │
│                                 │
│ ☐ 트리니티 4% 모드              │
└─────────────────────────────────┘
```

**물가상승률 시나리오**:
- **보수적 (3%)**: 낙관적 전망, 부족 금액 최소화
- **중립 (4%)**: 현실적 전망, 균형잡힌 분석
- **공격적 (5%)**: 비관적 전망, **"공포 수치" 극대화** ← 상담용 권장

**트리니티 4% 모드**:
- 체크 시: 원금 보존하며 연 4%만 인출 (안전 인출률)
- 미체크 시: 자산 고갈 방식 (수령액 최대화)
- 일반 상담: 미체크 권장 (수령액 높게 보임)

## 4. 상담 시나리오

### 🎯 시나리오 A: "공포 타격형" 상담

**목표**: 물가 공포를 활용한 추가 가입 유도

#### Step 1: 공격적 시나리오 선택
- 물가상승률: **공격적 (5%)** 선택
- 트리니티 모드: **미체크** (수령액 높게 보임)

#### Step 2: 분석 실행 후 태블릿 회전
- 태블릿을 고객 쪽으로 돌려 화면 공유
- 우측 패널의 **"생활비 Gap 분석"** 섹션 강조

#### Step 3: 공포 수치 강조
```
🔴 심각: 물가상승률 5% 반영 시

• 현재 최소 생활비: 310만원/월
• 25년 후 필요 생활비: 1,050만원/월
• 4층 보장 합계: 395만원/월
• 부족 금액: 655만원/월 (62.4%)
```

**멘트 예시**:
> "고객님, 이 수치는 국가통계포털 데이터를 기반으로 한 과학적 분석입니다.  
> 물가상승률 5%를 반영하면, 25년 후 지금의 310만원이 1,050만원이 필요합니다.  
> 현재 계획으로는 **월 655만원이 부족**합니다."

#### Step 4: 해결책 제시
- 4층 보장 바 차트 보여주며 "개인연금" 영역 강조
- "월 납입액을 50만원으로 늘리면 Gap이 절반으로 줄어듭니다"
- 세제 혜택 강조: "연 59만원 환급받으시면 실제 부담은 더 적습니다"

**목표**: 세액공제 혜택으로 가입 유도

#### Step 1: 세제 혜택 섹션 강조

**멘트 예시**:

## 5. FAQ

### Q1. 분석 결과가 너무 비관적으로 나오는데요?
**A**: 물가상승률 시나리오를 "중립(4%)" 또는 "보수적(3%)"으로 변경하세요. 다만, 상담 시에는 "공격적(5%)"을 권장합니다. 현실적인 위기감을 주는 것이 추가 가입 유도에 효과적입니다.

### Q2. 국민연금 예상액을 모르는 고객은 어떻게 하나요?
**A**: 국민연금공단 앱(내 곁에 국민연금)에서 즉시 조회 가능합니다. 앱 설치 후 공동인증서 로그인하면 예상 수령액이 나옵니다. 또는 평균값(월 172만원)을 입력하고 "정확한 금액은 조회 후 재분석하겠습니다"라고 안내하세요.

### Q3. 트리니티 4% 모드는 언제 사용하나요?
**A**: 고객이 "원금을 보존하고 싶다"고 하거나, 자녀에게 상속을 고려하는 경우 사용합니다. 일반 상담에서는 미체크를 권장합니다 (수령액이 높게 나와 고객 만족도가 높음).

### Q4. 분석 결과를 저장하거나 출력할 수 있나요?
**A**: 현재 버전에서는 화면 캡처를 권장합니다. 향후 업데이트에서 PDF 다운로드 및 카카오톡 발송 기능이 추가될 예정입니다.

### Q5. 모바일에서도 사용 가능한가요?
**A**: 네, 반응형 디자인이 적용되어 있습니다. 다만, 태블릿(10인치 이상)에서 사용하시는 것을 권장합니다. 스마트폰에서는 좌우 패널이 세로로 스태킹되어 표시됩니다.

## 6. 주의사항

### ⚠️ 필수 확인 사항

#### 1. 고객 동의
- 연금 분석 전 반드시 고객의 동의를 받으세요
- "국가 통계 기반 분석"임을 명확히 안내하세요
- 분석 결과는 "예상"이며 "확정"이 아님을 강조하세요

#### 2. 데이터 정확성
- 고객이 제공한 정보(나이, 소득, 연금액)의 정확성을 확인하세요
- 특히 국민연금 예상액은 반드시 공단 조회 결과를 사용하세요
- 추정값 사용 시 "정확한 조회 후 재분석 필요"를 안내하세요

#### 3. 물가상승률 설명
- "공격적(5%)" 시나리오는 비관적 전망임을 설명하세요
- "실제 물가상승률은 변동 가능"함을 안내하세요
- 고객이 불안해하면 "중립(4%)" 시나리오도 함께 보여주세요

#### 4. 세제 혜택 정확성
- 세액공제율은 연 총급여 기준으로 자동 계산됩니다
- 5,500만원 기준으로 16.5% / 13.2%가 적용됩니다
- 고객의 정확한 총급여를 입력해야 정확한 환급액이 나옵니다

#### 5. 유지율 경고
- 유지율 50%는 업계 평균이며, 개별 상품마다 다를 수 있습니다
- 중도 해지 시 불이익을 명확히 설명하세요
- "10년 만기 주머니 쪼개기" 전략을 적극 권장하세요
