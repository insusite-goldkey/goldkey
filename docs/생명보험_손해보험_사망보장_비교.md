# 생명보험 vs 손해보험 사망보장 비교 분석
> 판례 · 관련 법령 · 금융감독원(FSS) 표준약관 근거

---

## 🎯 보험종사자 실무 포인트 — 생명보험 사망상품 권장 이유

### (1) 사망 원인을 따지지 않는다
생명보험의 주계약 '일반사망'은 피보험자가 사망하기만 하면 **원인을 불문하고** 보험금을 지급합니다.
사인 미상·돌연사 시에도 분쟁 여지가 최소화됩니다.

#### (1-1) 가입 후 2년 경과 시 자살도 보장
- 상법 제659조에도 불구하고, 생명보험 표준약관은 **"보험계약일부터 2년이 지난 후 자살한 경우"** 보험금 지급을 명시
- 대법원 2007.9.6. 선고 **2006다15816 판결** 참조
- ⚠️ **법률 개정 시 개정 법률에 따름**

#### (1-2) 손해보험 사망보장의 구조적 한계
손해보험의 사망담보는 **상해사망**과 **질병사망**으로 구분됨.

| 구분 | 내용 |
|------|------|
| 상해사망 | 급격·우연·외래의 사고(상해 3요소) 충족 필요 |
| 질병사망 | 사망 원인이 **'상세불명'** 또는 **'의증'**인 경우 문제 발생 |

**질병사망 특약의 핵심 리스크:**
사망 원인이 '상세불명' 또는 '의증'으로 기재된 경우,
보험사는 사망에 이르게 한 질병 원인 확진을 위해 **검시(檢屍)** 요구 가능.
→ 유족 입장에서 심리적·절차적 부담이 매우 큼.

---

### (2) 중도 해지 또는 납입 완료 후 현금흐름이 좋다

| 구분 | 생명보험 | 손해보험 |
|------|----------|----------|
| 중도 해지 | 납입원금에 근접한 해지환급금 | 대부분 순수 소멸형 → 환급 거의 없음 |
| 납입만기 후 | 해약환급금 확보 가능 | 보험료 저렴하나 환급 없음 |
| 보험금 지급 안정성 | 높음 (원인 불문) | 낮음 (조건부 지급, 분쟁 多) |

> **손해보험 사망담보의 보험료가 저렴한 이유:**
> 지급 조건이 엄격하여 실제 지급률이 낮기 때문.
> 저렴한 대신 **보험금 지급 불안정** + **중도해지 환급 없음** 이라는 단점 존재.

## 📊 생명보험 vs 손해보험 사망보장 비교 총괄

| 구분 | 생명보험 (종신/정기보험) | 손해보험 (상해/질병사망) |
|------|------------------------|------------------------|
| 보장 범위 | 일반사망 (원인 불문, 모든 사망) | 상해사망 또는 질병사망 (제한적) |
| 주요 담보 | 주계약 (일반사망) | 특약 (상해사망, 질병사망) |
| 자살 보장 | 가입 2년 경과 시 지급 (특약 제외) | 원칙적 면책 (고의 사고) |
| 보장 기간 | 종신 (평생 보장 가능) | 최대 80세 또는 100세 (만기 존재) |
| 보험료 수준 | 상대적으로 높음 (지급 확률 100%) | 상대적으로 낮음 (조건부 지급) |

## 📌 생명보험 사망상품 장·단점

### [장점]
1. **포괄적 보장 범위** — 일반사망: 원인 불문 보험금 지급, 분쟁 최소화
2. **자살 보장 (2년 경과 시)** — 상법 제659조 적용 배제 (표준약관 기준)
3. **종신 보장** — 만기 없는 종신보험 형태 가능, 상속세 재원 마련 등 자산가에 유리

### [단점]
1. **높은 보험료** — 사망 시 반드시 지급해야 하므로 사차익 관리 비용 반영
2. **적립 구조** — 해지환급금 적립으로 사업비 비중 높을 수 있음

---

## 📌 손해보험 사망보장 장·단점

### [장점]
1. **저렴한 보험료** — 특정 사건(상해) 또는 질병에 의한 사망만 보장
2. **상해사망의 명확성** — 상해 3요소(급격·우연·외래) 충족 시 고액 보장 가능

### [단점]
1. **엄격한 입증 책임** — 상해 3요소 + 질병과의 인과관계 문제 시 분쟁 多
   - 대법원 2001.8.21. 선고 **2001다27579 판결** 참조
2. **질병사망의 한계** — 보험업법 시행령 제1조의2에 따라 만기 제한 (80세→최근 100세로 완화 추세), 종신 보장 불가
3. **자살 면책** — 고의 사고는 상법 및 약관에 따라 엄격 면책
   - 다만 심신상실 상태 자살은 예외적 인정 (입증 매우 까다로움)

## ⚖️ 관련 법조문 및 판례

| 법령/판례 | 내용 |
|-----------|------|
| **상법 제659조** | 고의·자살 면책 원칙 (생명보험 표준약관으로 2년 후 배제) |
| **상법 제732조** | 15세 미만자·심신상실자 사망보험계약 무효 (생·손보 공통) |
| **상법 제737조** | 손해보험 상해사망 기준 |
| **보험업법 제2조 제4호** | 손해보험에서 사망은 주계약 불가, 반드시 특약 형태로만 가입 가능 |
| **보험업법 시행령 제1조의2** | 손해보험 질병사망 특약 만기 제한 (80세 → 최근 100세로 완화 추세) |
| **대법원 2006다15816** | 생명보험 2년 경과 후 자살 보험금 지급 인정 |
| **대법원 2001다27579** | 상해 인정 요건 — 급격·우연·외래성 충족 필요 |
