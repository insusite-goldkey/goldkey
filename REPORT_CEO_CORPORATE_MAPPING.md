## 탭 구조 및 매핑

### [t7] 법인상담(CEO플랜) — 통합 법인 컨설팅 센터

**파일 위치**: `hq_app_impl.py` Line 51740~51953

#### 탭A: 🏢 법인상담(CEO플랜)

**핵심 화법**:
- **법인 감자플랜 화법** (상법 제343조)
  - "유가족이 받은 주식을 법인에 팔아 현금을 확보하는 나만의 비자금"
  - CEO 유고 시 → 법인 보험금 수령 → 상속인 주식 매입 → 소각
  - 잔존 주주 지분 가치 상승 + 상속인 유동성 확보 동시 해결
  - **상품 관점: 수익률이 아닌 자산의 안정적 이동 및 방어**

- **CEO 퇴직플랜 화법** (법인세법 시행령 §44)
  - "임원퇴직금 규정으로 법인에서 세금 없이 내 노후자금을 꺼내는 합법적 출구"
  - 퇴직소득세율(11~22%) vs 종합소득세율(15~45%) — 절세 효과 명확
  - 퇴직전 1년 급여 × 근속연수 × 3배 한도 손금
  - 경영인정기보험: 법인납입 → 사망보험금 → 퇴직금 재원 활용

**AI 분석 프롬프트**:
```
[법인상담 - CEO플랜]
임직원수: {emp_count}명, 법인자산: {corp_asset:,}만원
1. 법인 보험의 세무처리(손금산입) 방법
2. CEO 유고 시 법인 리스크 관리
3. 단체보험 가입 기준과 보장 설계
4. 퇴직금 재원 마련을 위한 보험 활용
5. 법인 감자플랜(상법§343) 및 CEO 퇴직플랜 화법 중심으로 분석
   — 상품은 수익률이 아닌 자산의 안정적 이동·방어 관점으로 설명
```

**연동 기능**:
- 🔥 공장화재보험 상담 바로가기 → `fire` 탭 이동

#### 탭B: 👑 VVIP CEO 전략센터

**연동**:
- 🚀 VVIP CEO 전략센터 전체 기능 열기 → `gk_sec09` 탭 이동
- 최근 비상장주식 평가 결과 자동 연동 (`ceo_eval_corp`, `ceo_eval_inh`)

**주요 기능 안내**:
- 📊 비상장주식 평가엔진 — 상증세법 §63 (순손익·순자산가치, 최대주주 할증 +30%)
- 🏛️ CEO 유고 상속세 시뮬레이터 — 과세표준·세율·예상세액 자동 산출
- 🔄 감자 플랜 시나리오 — 임직원 종신보험 + 자기주식 취득 시뮬레이션
- 🔥 법인 공장 화재보험 120% 안전규칙 + 최고위험요율 자동판정
- 🔍 재무제표 스캐너 — 가지급금·미처분이익잉여금 자동 감지
- 💰 중간배당 체크리스트 (상법 §462)

#### 탭C: 📊 기업·사업자 세무전략

**연동**:
- 🚀 기업·사업자 세무전략 전체 기능 열기 → `gk_sec08` 탭 이동

**법인 세무전략 핵심**:
- 경영인정기보험 손비처리: 연간 보험료 × 22% = 법인세 절감액
- 이익잉여금 → 보험료 충당 시 비상장주식 평가액 하락 효과
- 임원 퇴직금 한도: 1년 급여 × 근속연수 × 3배 (법인세법 §44)
- 가지급금 처리 7가지: 상여·유상증자·대물변제·퇴직금상계·보험충당·채권포기·매각대금
- 미처분이익잉여금 과다 → 중간배당·CEO퇴직금·유상증자로 조기 처분

**개인사업자 세무전략 핵심**:
- 복식부기 의무자: 화재보험료·단체상해보험료 전액 필요경비 인정
- 추계신고: 기준경비율 내 포함 — 별도 인정 불가
- 종합소득세 최고세율: 45%(10억 초과) vs 퇴직소득세 최대 22%
- 화재보험료 필요경비 인정 시 절세 = 보험료 × 적용 세율
- 사업주 본인 상해보험은 필요경비 불인정 (개인 비용)

#### 탭D: 💼 단체보험 설계

**AI 분석 프롬프트**:
```
[단체보험 설계]
임직원수: {emp_count2}명
1. 단체상해보험 가입 기준과 보장 범위
2. 산재보험과의 GAP 분석
3. 단체생명보험 설계 전략
4. 복리후생 설계와 세무처리
```

**핵심 포인트**:
- 임직원 5인 이상 시 단체보험 가입 유리
- 업무상 상해·질병 보장 포함
- 산재보험 GAP Cover 설계 권장
- 보험료 전액 손금산입 가능

#### 탭E: 📈 비상장주식평가

**연동**:
- 🚀 CEO플랜 비상장주식 평가 (t8) → `t8` 탭 이동
- 📊 비상장주식 평가 센터 (stock_eval) → `stock_eval` 탭 이동

**비상장주식 평가 공식** (상증세법 제63조):
```
주당 가치 = (순손익가치 × 3 + 순자산가치 × 2) ÷ 5
순손익가치 = 최근 3년 평균 순이익 ÷ 자본환원율(10%)
최대주주 할증: +30% (2024년~, 중소기업 +15%)
```

**전략**:
- 가업승계 전략: 평가액 조정을 통한 절세 시뮬레이션
- 경영권 방어: 상법 §341 자기주식 취득 → 소각 → 잔존주주 지분 상승

**파일 위치**: `hq_app_impl.py` Line 51956~52200+

#### 분석 방식 선택

**1. 📊 직접 입력 평가표**

**입력 항목**:
- 법인명
- 발행주식 총수 (주)
- 최대주주 여부 (경영권 할증 20% 적용)
- 부동산 과다 법인 여부 (자산 비중 80% 이상)
- 매매사례가액 (원, 없으면 0)
- 순자산 (원)
- 당기순이익 3개년 (최근년, 전년, 전전년)

**평가 엔진**: `AdvancedStockEvaluator`
- `evaluate_corporate_tax()`: 법인세법상 시가 산출
- `evaluate_inheritance_tax()`: 상증법상 최종가액 산출

**세션 저장**:
- `ceo_eval_corp`: 법인세법상 평가 결과
- `ceo_eval_inh`: 상증법상 평가 결과
- `ceo_company_result`: 법인명
- `ceo_shares_result`: 발행주식 총수

**AI 심층 분석**: `CEO_PLAN_PROMPT` 사용

**2. 📁 재무제표 스캔 업로드**

**업로드 파일**: PDF, JPG, JPEG, PNG (다중 업로드 가능)

**GP193 전역 인제스트 후크**:
- 업로드 즉시 GCS + RAG 동시 저장
- `source_tab`: `ceo_plan_fs`
- `doc_type`: `기타`

**AI 분석**: `CEO_FS_PROMPT` 사용
