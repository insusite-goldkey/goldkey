## 2️⃣ 핵심 로직 구현

### A. 80% 코인슈어런스 (Co-insurance Clause)

**정의**: 재물보험에서 사고 발생 시 실제 손해액 전액을 보상받기 위한 부보 비율 규칙

**의무 가입 기준**: 건물 가액(보험가액)의 **80% 이상** 설정 필수

**비례 보상 공식**:
$$
\text{보험금} = \text{손해액} \times \frac{\text{실제 가입금액}}{\text{보험가액} \times 80\%}
$$

**핵심 포인트**:
- 이는 단순 본인부담금이 아니라, 충분한 담보를 설정하지 않은 가입자에게 부과하는 **'과소 보험 패널티'**
- 보험가액의 80% 이상 가입 시: **실손보상** (손해액 전액)
- 80% 미만 가입 시: **비례보상** (패널티 적용)

**구현 코드** (`property_engine.py:291-358`):
```python
def calculate_coinsurance_payout(
    actual_cash_value: float,
    insured_amount: float,
    loss_amount: float,
) -> CoinsuranceResult:
    # 80% 기준 최소 가입액
    coinsurance_minimum = actual_cash_value * COINSURANCE_RATIO
    
    # 충분 가입 여부 판단
    is_adequate = insured_amount >= coinsurance_minimum
    
    if is_adequate:
        # 실손보상: 손해액 전액 지급
        actual_payout = min(loss_amount, insured_amount)
        penalty_ratio = 1.0
        penalty_amount = 0
    else:
        # 비례보상: 패널티 적용
        penalty_ratio = insured_amount / coinsurance_minimum
        actual_payout = loss_amount * penalty_ratio
        penalty_amount = loss_amount - actual_payout
```

### B. 재조달가액 vs 보험가액

**1. 핵심 개념 정의**

| 구분 | 정의 | 핵심 포인트 |
|------|------|-------------|
| **재조달가액 (Replacement Cost)** | 보험사고 발생 시점과 장소에서, 해당 건물을 신규로 다시 짓거나 구입하는 데 드는 비용 | "새것(New)" 기준 가격 / 물가상승 반영 |
| **보험가액 (Actual Cash Value)** | 재조달가액에서 사용에 따른 경년감가액을 공제한 현재의 실질적 가치 | 보상할 수 있는 법적 최고 한도 |

**2. 보험가액 산출 공식**

$$
\text{보험가액} = \text{재조달가액(사고시점)} - \text{경년감가액}
$$

$$
\text{경년감가액} = \text{재조달가액} \times \text{경과년수} \times \text{연간 감가율}
$$

**시사점**: 건물이 오래될수록 보험가액(보상 한도)은 낮아지며, 이를 무시하고 보험가입금액만 높게 잡으면 **'초과보험'**이 되어 보험료만 낭비하게 됩니다.

**구현 코드** (`property_engine.py:234-284`):
```python
def calculate_asset_valuation(
    replacement_cost: float,
    building_year: int,
    current_year: int,
    structure: str,
) -> AssetValuation:
    # 건축 경과년수 계산
    building_age = current_year - building_year
    
    # 구조별 감가율 조회
    depreciation_rate = DEPRECIATION_RATES.get(structure, 0.015)
    
    # 총 감가액 계산
    total_depreciation_ratio = min(building_age * depreciation_rate, 1.0)
    total_depreciation = replacement_cost * total_depreciation_ratio
    
    # 보험가액 (시가) 계산
    actual_cash_value = replacement_cost - total_depreciation
    
    # 80% 코인슈어런스 최소 가입액
    coinsurance_minimum = actual_cash_value * COINSURANCE_RATIO
```

### C. 감가상각 엔진 (Depreciation Engine)

**구조별 연간 감가율**:

| 구조 | 연간 감가율 | 비고 |
|------|-------------|------|
| **내화구조** (콘크리트, 철근콘크리트) | 1.0% | 가장 낮은 감가율 |
| **준내화구조** (경량철골, ALC) | 1.5% | 표준 감가율 |
| **철골구조** | 1.2% | 내화구조와 준내화구조 중간 |
| **목구조** | 2.0% | 높은 감가율 |
| **가설구조** (컨테이너, 조립식) | 3.0% | 가장 높은 감가율 |

**구현 코드** (`property_engine.py:50-60`):
```python
DEPRECIATION_RATES = {

    STRUCTURE_STEEL: 0.012,          # 철골구조 1.2%/년
    STRUCTURE_WOOD: 0.020,           # 목구조 2.0%/년
```
